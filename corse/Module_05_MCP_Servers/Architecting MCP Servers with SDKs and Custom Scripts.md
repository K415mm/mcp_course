Building Model Context Protocol (MCP) servers demonstrates the flexibility of the protocol, as they can be written in any programming language and used with any modern agentic AI framework 1\. The source material provides a detailed look at two different approaches to building an MCP server: using an official SDK (Python) and building one from scratch (Bash). \#\#\# 1\. Building an MCP Server using an SDK (Python Filesystem Server) The easiest and most recommended way to build an MCP server is by using an official SDK, such as the MCP Python SDK 2, 3\. The SDK's FastMCP class handles all the heavy lifting, such as registering tools, negotiating the protocol version, and exposing them via MCP transports 2\. To expose a function as a tool, you simply use the @mcp.tool() decorator 4\.
